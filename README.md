# CodeAlpha Stock Portfolio Tracker

A simple Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Python Programming Internship.

## Features

- View available stocks and their prices
- Add stocks to your portfolio
- Update existing stock holdings
- Remove stocks from portfolio
- Calculate total investment value
- Export portfolio data to a CSV file

## Technologies Used

- Python 3
- CSV Module
- Datetime Module

## Available Stocks

- AAPL
- TSLA
- GOOGL
- AMZN
- MSFT
- NFLX
- META
- NVDA

## How to Run

Open a terminal in the project directory and execute:

```bash
python DHL_stock_tracker.py
```

## Sample Operations

- Add a stock
- View portfolio summary
- Remove a stock
- Save portfolio to CSV
